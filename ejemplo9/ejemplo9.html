Problema 9: Uso incorrecto de atributos ARIA


X

Solución 9:



Explicación:

En lugar de usar un <div> con role="button", es mejor usar un elemento <button> real. Esto proporciona funcionalidad de botón nativa y mejora la accesibilidad sin necesidad de JavaScript adicional. Los elementos <button> ofrecen ventajas inherentes como el enfoque del teclado, la activación con la tecla Enter, y una semántica clara para tecnologías de asistencia, lo que hace que tu sitio web sea más inclusivo y fácil de usar para todos los usuarios.
